# NOVARO Menswear — Advanced E-Commerce Front-End

NOVARO Menswear is a premium-but-affordable men's clothing e-commerce website concept. It is inspired by the clean American fashion feeling of brands like Tommy Hilfiger, Guess and Calvin Klein, but uses an original brand name, logo, products and design.

## Live Demo

After uploading to GitHub Pages, your site will be available at:

```text
https://YOUR-USERNAME.github.io/novaro-menswear/
```

## Features

- Premium original brand identity: **NOVARO Menswear**
- Responsive homepage with hero section
- Product collection with 9 clothing/accessory items
- Search by product name, category or description
- Category filter
- Price range filter
- Sort by featured, low price, high price and rating
- Wishlist using localStorage
- Shopping cart drawer
- Quantity increase/decrease controls
- Size selection
- Automatic 10% bundle discount when cart includes shirt + denim + jacket
- Free shipping over $75
- Checkout UI mockup with payment method options
- Newsletter form
- Social media links
- Dark mode toggle
- Mobile navigation menu
- Portfolio-ready responsive design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
- GitHub Pages compatible

## Important Note About Payments

The payment section is a front-end demo only. It does **not** process real payments. To make it real, connect Stripe, PayPal, Shopify, WooCommerce, Firebase, Supabase or another backend/payment provider.

## How to Upload to GitHub

1. Create a new repository named `novaro-menswear`.
2. Upload these files to the repository root, not inside another folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose `main` branch and `/root` folder.
6. Click **Save**.
7. Wait 1–3 minutes and open your GitHub Pages link.
